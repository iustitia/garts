About the project

Gender of german articles is not an easy thing to learn. This project helps in learning it, by letting you use quizes to memorize them.

Project stack: Application is written in Python with Django framework. Quiz logic is based on 'django_quiz' with modifications and extensions.

The whole project is open-source. You can find code at GitHub: Garts

Source for word list for this app is Tatoeba, sorted by popularity.

You can use this site in personal and commercial purposes.
